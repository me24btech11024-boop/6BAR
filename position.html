Position Analysis & Vector Loop Closure

The 4-bar crank-rocker + dyad architecture relies on two interdependent complex vector loops to mathematically verify the position geometry across the stroke:

  1. Primary 4-Bar Loop: R2 + R3 - R4 - R1 = 0 (Crank + Coupler 1 - Rocker - Ground)
  2. Secondary Dyad Loop: R4 + R5 - R_out - R_offset = 0 (Rocker + Coupler 2 - Vertical Output - Ground Offset)
